# Runtime Foundations

Advanced reference.

Runtime foundations are built-in runtime baseline bundles for AI engineering environments.

- `foundations/` declares which profile and overlays belong together
- `profiles/` provides the editable baseline copied into `.praxis/foundation/profile/`
- `overlays/` provides extension seams copied into `.praxis/overlays/`

Most users do not need to read these docs during initial setup. Praxis applies the built-in runtime baseline automatically on the default install path.
