# Runtime Foundations

Runtime foundations are built-in baseline bundles for AI engineering environments.

- `foundations/` declares which profile and overlays belong together.
- `profiles/` provides the editable baseline copied into `.praxis/foundation/profile/`.
- `overlays/` provides extension seams copied into `.praxis/overlays/`.

Unlike `stacks/`, foundations describe the runtime operating model for AI engineering workflows, not language-specific toolchains.
