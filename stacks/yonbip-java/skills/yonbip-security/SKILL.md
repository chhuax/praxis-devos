# YonBIP 安全编码规范（栈专属扩展）

> 本 skill 是 `security`（通用）的 **YonBIP 栈扩展**。通用安全编码规范（OWASP）请参考通用 skill。

## 1. 安全扫描工具链

### 1.1 必须使用的工具

| 扫描类型 | 工具 | 要求 |
|---------|------|------|
| 代码扫描（SAST） | CheckMarx / YCG | 中危及以上漏洞必须修复 |
| 黑盒扫描（DAST） | 安全测试工具 | 中危及以上漏洞必须修复 |
| 渗透测试 | 安全团队 | 发现漏洞必须修复 |

### 1.2 开源组件管理

```
❌ 禁止使用有已知漏洞的组件
❌ 禁止使用 EOM 后的组件
❌ 禁止 fastjson（升级到安全版本或替换）

✅ 使用自动扫描工具检查漏洞
```

---

## 2. YonBIP 密码策略

### 2.1 密码复杂度

```
长度：≥8位
包含：大写字母、小写字母、数字、特殊字符（≥3种）
禁止：包含手机号、企业名称、用户名
有效期：90天
```

### 2.2 密码传输

```
❌ 明文传输
❌ MD5/SHA1
❌ Base64

✅ RSA
✅ SM4
✅ 加盐哈希
```

---

## 3. YonBIP 会话策略

### 3.1 会话超时

```
无活动超时：≤10分钟
绝对超时：3小时（Web端）
```

### 3.2 登录失败处理

```
失败次数：5次
锁定策略：1小时锁定或管理员解锁
失败示警：剩余次数提示
```

---

## 4. YonBIP 脱敏规则

| 信息类型 | 脱敏规则 |
|---------|---------|
| 身份证 | 显示首尾位，如：3\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*1 |
| 手机号 | 隐藏中间6位，如：134\*\*\*\*\*\*\*\*48 |
| 银行卡 | 仅显示后4位，如：\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*8639 |
| 地址 | 最多到区级 |
