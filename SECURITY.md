## Reporting a Vulnerability

We take security seriously. Please do not report security vulnerabilities
through public GitHub issues. Instead, send an email to [INSERT EMAIL].

### Response Timeline

* **48 hours**: Initial triage and acknowledgment.
* **7 days**: Full assessment and severity rating.
* **30 days**: Target for releasing a fix or mitigation.
