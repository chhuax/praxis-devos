## 关键差异项

- 配置差异
- 数据差异
- 外部依赖差异
- 权限与访问控制差异
- 监控与告警差异

## 使用规则

- 排障前必须先确认当前环境
- 任何命令、日志入口、监控入口都应标明适用环境
- 不得默认将生产环境行为套用到测试环境
