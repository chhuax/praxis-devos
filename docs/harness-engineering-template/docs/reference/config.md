# Config Reference

## Source Of Truth

- `contracts/config/app.schema.json`

## Scope

This document summarizes supported runtime configuration.

## Config Items

### `PORT`

- Type: integer
- Default: `3000`
- Purpose: HTTP listen port

### `LOG_LEVEL`

- Type: string
- Default: `info`
- Allowed values: `debug`, `info`, `warn`, `error`
