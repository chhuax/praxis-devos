# Getting Started

## Audience

Developers integrating with or contributing to the project.

## Preconditions

- Required runtime installed
- Project dependencies installed
- Valid credentials if the project talks to external systems

## Steps

1. Install dependencies.
2. Start the application locally.
3. Run the health check endpoint or equivalent smoke test.
4. Run the test suite.
5. Read `AGENTS.md` before making changes.
